# Publish This Site

The site files are ready in this folder:

- `index.html`
- `styles.css`
- `script.js`
- `.nojekyll`
- `README.md`

## Upload Through GitHub

1. Open `https://github.com/KizzyMoon/HighlifeRP`.
2. Choose **Add file**.
3. Upload the files listed above.
4. Commit them to the `main` branch.
5. Go to **Settings > Pages**.
6. Set the source to **Deploy from a branch**.
7. Choose `main` and `/root`.

The page should publish at:

`https://kizzymoon.github.io/HighlifeRP/`

## Important Note

The site's add buttons save new entries in your browser only. To make entries visible to everyone on GitHub Pages, add them to the starter data inside `script.js`.
